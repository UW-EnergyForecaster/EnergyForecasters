<li>To determine instrument measurement uncertainty, please download the <a target="_top" href="../radiometer_uncert.xlsx">Excel&reg; uncertainty spreadsheet</a> to compute these values. (5.5 MB XLSX)
<li>Further instrument information, including current serial numbers and calibration factors, are available by clicking the instrument name.  Some of these serial numbers can also be clicked for further instrument calibration history and uncertainties.
<li><i>Irradiance</i> data will always appear on primary axis and <i>meteorological</i> data will always appear on secondary axis.
<li><i>Selected data</i> or <i>all raw data</i> has the option of being compressed using the ZIP format.   An extraction utility can be obtained from <a href="http://www.pkware.com" target="_top">PKWare</a> or <a href="http://www.winzip.com" target="_top">WinZip</a>
<li><i>Selected hourly data</i> will generate hourly data by averaging the input data for the entire day(s), <i>custom data</i> will be ignored.  Note:  DO NOT use this feature with wind direction measurements, the resulting averages will be bogus.<li><i>Selected daily statistics</i> will generate daily totals (for <i>irradiance</i> data) and daily averages (for <i>meteorological</i> data) using the input data for the entire day(s), <i>custom data</i> will be ignored.  The daily maximum and minimum (if applicable) will also be generated.  Note: Negative values from shortwave irradiance instruments will be set to zero when accumulating daily totals.  DO NOT use this feature with wind direction measurements, the resulting averages will be bogus.<li>The <i>raw data</i> options are the complete raw data record(s) from the datalogger, these are the <a href="rawid.pl?site=VTIF;start=20120401;yr=0;mo=3;dy=0">raw data field IDs</a>.
<li><i>"Live" raw data</i> is a single record from the most recent data collection and <i>all raw data</i> is all the data from midnight to the latest record.<li><i>Selected display</i> will generate a customized data display, similar to the above table, <i>custom data</i> will be ignored.<li><i>English conversion</i> only applies to <i>selected data</i> or <i>plot</i> and has no effect on <i>irradiance</i> or <i>custom data</i>.
<li>When selecting <i>raw data</i> or <i>wind rose</i>, all selections (except the date) will be ignored.
<li>The <i>wind rose</i> is generated using average wind speed and direction for the selected level.
<li>The <i>calculate third component</i> option uses the formula: <i>Global</i> = <i>Direct</i> * Cosine(<i>Zenith</i>) + <i>Diffuse</i> to calculate a third component based on the selected two.  This option should only be used for pyrheliometers and horizontal pyranometers.
<li>For automated data extraction, please use the <a href="data_api_doc.pl?VTIF">MIDC API</a>.</ul>
